Remote Desktop Application

About

The Remote Desktop Application allows users to remotely access and control the desktop of a server machine over a network connection.

Server Side

The server side of the Remote Desktop Application is responsible for managing client connections, authenticating users, capturing the server's screen, and sending screen updates to connected clients.

                    
                        // Sample code to execute the server JAR file
                        java -jar server.jar

Client Side

The client side of the Remote Desktop Application is responsible for authenticating users, displaying the server's desktop interface, and handling user input events.

                    
                        // Sample code to execute the client JAR file
                        java -jar client.jar

Steps to Deploy

  1. Download the server and client JAR files from the repository.
  2. Execute the server JAR file on the server machine.
  3. Execute the client JAR file on the client machine.
  4. Enter the server's IP address and the password set for authentication.
  5. Access the remote desktop interface and start controlling the server's desktop remotely.

Dependencies

No external dependencies required. The JAR files contain all necessary dependencies bundled within them.
